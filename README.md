
# React Dashboard App

## 🚀 Features
- Dashboard with analytics cards (Users, Revenue, Orders, Growth)
- Users page with search and filtering
- Reusable table component
- Reports page with dynamic filtering
- Custom hook for API handling (loading, error, data)
- Debounced search for performance optimization

## 🛠 Tech Stack
- React (Vite)
- TypeScript
- React Router
- Axios

## 🧠 Key Concepts Used
- Custom Hooks
- Component Reusability
- State Management (useState, useEffect)
- Performance Optimization (debounce, useMemo)

## ▶️ Run Locally
npm install  
npm run dev
